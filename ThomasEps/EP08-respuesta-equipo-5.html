Pregunta 1

La frecuencia de personas mayores de 80 aos en la regin metropolitana Es similar en las dems regiones?

Desarrollo de actividades para resolver la pregunta:

Se utilizar la prueba de permutaciones (Monte Carlo) para comparar una variable continua en dos muestras independientes.

1). Obtencion de datos e hiptesis a contrastar:

# Carga de datos
casen <- read.csv("EP08 Datos CASEN 2017.csv")

# Limpiar y clasificar personas por regin y edad
casen_filtrado <- casen %>%
  filter(!is.na(edad), edad >= 0) %>%
  mutate(
    grupo_region= if_else(region == "Regin Metropolitana de Santiago", 
                           "Metropolitana", "Resto"),
    anciano = edad >= 80
  )

# Obtener IDs nicos de hogares presentes en los datos filtrados
ids_todos <- unique(casen_filtrado$id.vivienda)

# Fijar semilla y muestrear 120 hogares
set.seed(777)
ids_muestra <- sample(ids_todos, size = 120)

# Filtrar las personas pertenecientes a esos hogares seleccionados
muestra <- casen_filtrado %>%
  filter(id.vivienda %in% ids_muestra) %>%
  select(id.vivienda, grupo_region, edad, anciano)

Las hiptesis a contrastar son:

  • Hiptesis nula (H): La frecuencia de personas mayores de 80 aos en la regin metropolitana es similar en las dems regiones.

  • Hiptesis alternativa (Ha): La frecuencia de personas mayores de 80 aos en la regin metropolitana no es similar en las dems regiones.

Resumen de las muestras tomadas, teniendo en cuenta la cantidad de personas tomadas, de ancianos y de su proporcin en la Regin Metropolitana y del resto de regiones. Seguidamente se incluye la diferencia observada en la proporcin de personas mayores de 80 aos entre la Regin Metropolitana y el resto del pas. Para ello, se estim la proporcin de personas 80 aos en cada grupo y se obtuvo su diferencia (prop_dif), que corresponde al estadstico observado que ser contrastado mediante permutaciones.

# Resumen rpido
muestra %>%
  group_by(grupo_region) %>%
  summarise(
    personas  = n(),
    ancianos  = sum(anciano),
    proporcion = mean(anciano)
  )

## 1 Metropolitana       10        4     0.4

  muestra %>%

2). Se crea una gran cantidad P de permutaciones a partir de las muestras originales, usando muestreo sin reposicin sobre la muestra combinada, adems de obtener el p-value.

Tambin se incluye un histograma de la distribucin nula obtenida mediante permutaciones, el cual permitir visualizar la posicin del estadstico observado respecto a dicha distribucin.

set.seed(777)
# Permutaciones
B <- 4999
perm_diffs <- replicate(B, {
  grupo_region_perm <- sample(muestra$grupo_region)
  mean(muestra$anciano[grupo_region_perm == "Metropolitana"]) -
    mean(muestra$anciano[grupo_region_perm == "Resto"])
})

p1 <- gghistogram(
  data.frame(perm_diffs), "perm_diffs",
  bins = 40,
  fill = "blue",
  title = "Distribucin",
  xlab = "Diferencia entre proporciones de mayores de 80 aos (Metropolitana  Resto)",
  ylab = "Frecuencia usada"
)
p1 <- p1 + geom_vline(xintercept = prop_dif, colour = "red")
print(p1)

# Valorp bilateral
p_value <- mean(abs(perm_diffs) >= abs(prop_dif))
print(p_value)
## [1] 0.00140028

Conclusin:

Como el p-valor es menor al nivel de significancia (0.0014 < 0.05), se rechaza la hiptesis nula, por ende, con un 95% de nivel de confianza podemos decir que la frecuencia de personas mayores de 80 aos en la regin metropolitana no es similar en las dems regiones.

Pregunta 2

La edad promedio de las personas mayores de 18 aos Es similar en los estados de soltero, casado y viudo?

  • Hiptesis nula (H): Las medias poblacionales de edad son iguales en los tres grupos de estado civil.

  • Hiptesis alternativa (Ha): Al menos una de las medias poblacionales de edad difiere entre los grupos.

Desarrollo de actividades para resolver la pregunta:

Se utilizar bootstrapping con remuestras en conjunto con ANOVA para contestar la pregunta planteada

1). Obtencion de datos e hiptesis a contrastar:

Con el fin de simular correctamente la hiptesis nula de igualdad de medias entre los grupos de estado civil, se realizo un reescalado de las edades dentro de cada grupo. Dicho procedimiento consisti en ajustar los valores de edad para que cada grupo tuviera la misma media y desviacin estndar globales, manteniendo al mismo tiempo la estructura relativa interna de cada grupo. De esta forma, se genera un conjunto de datos artificial donde las diferencias entre grupos han sido eliminadas, permitiendo aplicar el mtodo bootstrap de manera vlida para contrastar la hiptesis nula.

grupos <- muestra2 |>
  group_by(ecivil) |>
  summarise(
    media = mean(edad),
    sd = sd(edad)
  ) |>
  as.data.frame()

# Calcular media y desviacin estndar global
media_gral <- mean(muestra2$edad)
sd_gral <- sd(muestra2$edad)


solteros <- muestra2 %>%
  filter(ecivil == "Soltero(a)") %>%
  mutate(edad = media_gral +
           (edad - mean(edad)) * (sd_gral / sd(edad)))

casados <- muestra2 %>%
  filter(ecivil == "Casado(a)") %>%
  mutate(edad = media_gral +
           (edad - mean(edad)) * (sd_gral / sd(edad)))

viudos <- muestra2 %>%
  filter(ecivil == "Viudo(a)") %>%
  mutate(edad = media_gral +
           (edad - mean(edad)) * (sd_gral / sd(edad)))

# Unir todos los grupos reescalados
muestra2_Z <- bind_rows(solteros, casados, viudos)

Ahora se generan las muestras bootstrap para cada grupo. Obteniendo valores de F para cada una de las muestras generadas, adems se incluye un grfico de la distribucin de los valores de F obtenidos a partir de las muestras bootstrap, junto con el valor observado del estadstico F para obtener una comparacin visual.

# Generacion de muestras bootstrap
set.seed(573)
B <- 2999

solteros <- muestra2_Z %>%
  filter(ecivil == "Soltero(a)") %>%
  select(edad, ecivil)

casados <- muestra2_Z %>%
  filter(ecivil == "Casado(a)") %>%
  select(edad, ecivil)

viudos <- muestra2_Z %>%
  filter(ecivil == "Viudo(a)") %>%
  select(edad, ecivil)


re_sol <- lapply(1:B, function(i) sample(nrow(solteros), replace = TRUE))
re_cas <- lapply(1:B, function(i) sample(nrow(casados),  replace = TRUE))
re_viu <- lapply(1:B, function(i) sample(nrow(viudos),  replace = TRUE))


# Funcin bootstrap para calcular el estadstico F
bootstrap_f <- function(b) {
  datos_b <- rbind(
    solteros[re_sol[[b]], ],
    casados[re_cas[[b]], ],
    viudos[re_viu[[b]], ]
  )
  
  modelo <- aov(edad ~ ecivil, data = datos_b)
  car::Anova(modelo, type = 2)$`F value`[1]
}



boot_F <- sapply(1:B, bootstrap_f)


p2 <- gghistogram(data.frame(boot_F), x = "boot_F",
                  title = "Distribucin permutada",
                  xlab = "Estadstico F", ylab = "Frecuencia",
                  bins = 40, fill = "blue")
p2 <- p2 + geom_vline(xintercept = F_obs, colour="red")
print(p2)

En el grfico,la lnea roja vertical indica el valor de F observado en los datos reales. Dado que este valor se encuentra claramente fuera del rango esperado bajo la hiptesis nula, se estima que la diferencia observada entre los grupos es estadsticamente significativa.

Ahora se realizar el contraste de hiptesis, calculando el valor crtico y el p-valor emprico bootstrap. El valor crtico se obtiene a partir del percentil 95 de la distribucin nula generada por el bootstrap, mientras que el p-valor emprico se calcula como la proporcin de valores de F en la distribucin nula que son mayores que el valor observado:

F_crit <- quantile(boot_F, 0.95)
num2 <- sum(boot_F > F_obs)
p_val2 <- (num2 + 1) / (B + 1)

print(F_crit)
##      95% 
## 3.121253
print(p_val2)
## [1] 0.0003333333

Como resultado, se obtuvo un valor crtico F igual a 3.12 (percentil 95% de la distribucin nula), mientras que el valor del estadstico F observado fue superior a dicho umbral. Por otro lado el p-valor fue menor al nivel de significancia (p < 0.05), por lo tanto, se concluye con un 95% de confianza que existen diferencias significativas en la edad promedio entre al menos dos de los grupos de estado civil analizados (soltero, casado y viudo).

POST-HOC

En este caso, el resultado de la prueba fue estadsticamente significativo (p < 0.05), se realiza entonces el anlisis post-hoc con fines acadmicos, para explorar posibles diferencias y ejercitar la metodologa. Utilizando la diferencia de las medias como nuevo estadstico de inters.

Para cada par de grupos (soltero-casado, soltero-viudo, casado-viudo), se calcul la diferencia observada entre las medias de edad, y se gener una distribucin nula de diferencias mediante 2999 remuestras bootstrap con reemplazo. A partir de estas distribuciones, se calcularon los valores-p empricos bilaterales, los cuales daran un indicio para saber si las diferencias observadas son significativamente distintas de cero.

dif_obs_SC <- mean(solteros$edad) - mean(casados$edad)
dif_obs_SV <- mean(solteros$edad) - mean(viudos$edad)
dif_obs_CV <- mean(casados$edad)  - mean(viudos$edad)

get_dif_medias <- function(grupo1, grupo2, indices1, indices2) {
  media1 <- mean(grupo1[["edad"]][indices1])
  media2 <- mean(grupo2[["edad"]][indices2])
  media1 - media2
}

# Comparacin Solteros - Casados
boot_dif_SC <- sapply(1:B, function(b) {
  get_dif_medias(solteros, casados, re_sol[[b]], re_cas[[b]])
})

# Comparacin Solteros - Viudos
boot_dif_SV <- sapply(1:B, function(b) {
  get_dif_medias(solteros, viudos, re_sol[[b]], re_viu[[b]])
})

# Comparacin Casados - Viudos
boot_dif_CV <- sapply(1:B, function(b) {
  get_dif_medias(casados, viudos, re_cas[[b]], re_viu[[b]])
})


# Diferencia observada y p-valor emprico bootstrap (dos colas)
dif_obs_SC <- mean(solteros$edad) - mean(casados$edad)
p_SC <- mean(abs(boot_dif_SC) >= abs(dif_obs_SC))
dif_obs_SV <- mean(solteros$edad) - mean(viudos$edad)
p_SV <- mean(abs(boot_dif_SV) >= abs(dif_obs_SV))
dif_obs_CV <- mean(casados$edad) - mean(viudos$edad)
p_CV <- mean(abs(boot_dif_CV) >= abs(dif_obs_CV))
