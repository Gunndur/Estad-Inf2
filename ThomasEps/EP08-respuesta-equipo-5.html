Pregunta 1

La frecuencia de personas mayores de 80 años en la región metropolitana ¿Es similar en las demás regiones?

Desarrollo de actividades para resolver la pregunta:

Se utilizará la prueba de permutaciones (Monte Carlo) para comparar una variable continua en dos muestras independientes.

1). Obtencion de datos e hipótesis a contrastar:

# Carga de datos
casen <- read.csv("EP08 Datos CASEN 2017.csv")

# Limpiar y clasificar personas por región y edad
casen_filtrado <- casen %>%
  filter(!is.na(edad), edad >= 0) %>%
  mutate(
    grupo_region= if_else(region == "Región Metropolitana de Santiago", 
                           "Metropolitana", "Resto"),
    anciano = edad >= 80
  )

# Obtener IDs únicos de hogares presentes en los datos filtrados
ids_todos <- unique(casen_filtrado$id.vivienda)

# Fijar semilla y muestrear 120 hogares
set.seed(777)
ids_muestra <- sample(ids_todos, size = 120)

# Filtrar las personas pertenecientes a esos hogares seleccionados
muestra <- casen_filtrado %>%
  filter(id.vivienda %in% ids_muestra) %>%
  select(id.vivienda, grupo_region, edad, anciano)

Las hipótesis a contrastar son:

  • Hipótesis nula (H₀): La frecuencia de personas mayores de 80 años en la región metropolitana es similar en las demás regiones.

  • Hipótesis alternativa (Ha): La frecuencia de personas mayores de 80 años en la región metropolitana no es similar en las demás regiones.

Resumen de las muestras tomadas, teniendo en cuenta la cantidad de personas tomadas, de ancianos y de su proporción en la Región Metropolitana y del resto de regiones. Seguidamente se incluye la diferencia observada en la proporción de personas mayores de 80 años entre la Región Metropolitana y el resto del país. Para ello, se estimó la proporción de personas ≥80 años en cada grupo y se obtuvo su diferencia (prop_dif), que corresponde al estadístico observado que será contrastado mediante permutaciones.

# Resumen rápido
muestra %>%
  group_by(grupo_region) %>%
  summarise(
    personas  = n(),
    ancianos  = sum(anciano),
    proporcion = mean(anciano)
  )
## # A tibble: 2 × 4
##   grupo_region  personas ancianos proporcion
##   <chr>            <int>    <int>      <dbl>
## 1 Metropolitana       10        4     0.4   
## 2 Resto              138        5     0.0362
prop_dif <- diff(
  muestra %>%
    group_by(grupo_region) %>%
    summarise(prop = mean(anciano)) %>%
    pull(prop)
)
print(prop_dif)
## [1] -0.3637681

2). Se crea una gran cantidad P de permutaciones a partir de las muestras originales, usando muestreo sin reposición sobre la muestra combinada, además de obtener el p-value.

También se incluye un histograma de la distribución nula obtenida mediante permutaciones, el cual permitirá visualizar la posición del estadístico observado respecto a dicha distribución.

set.seed(777)
# Permutaciones
B <- 4999
perm_diffs <- replicate(B, {
  grupo_region_perm <- sample(muestra$grupo_region)
  mean(muestra$anciano[grupo_region_perm == "Metropolitana"]) -
    mean(muestra$anciano[grupo_region_perm == "Resto"])
})

p1 <- gghistogram(
  data.frame(perm_diffs), "perm_diffs",
  bins = 40,
  fill = "blue",
  title = "Distribución",
  xlab = "Diferencia entre proporciones de mayores de 80 años (Metropolitana − Resto)",
  ylab = "Frecuencia usada"
)
p1 <- p1 + geom_vline(xintercept = prop_dif, colour = "red")
print(p1)

# Valor‑p bilateral
p_value <- mean(abs(perm_diffs) >= abs(prop_dif))
print(p_value)
## [1] 0.00140028

Conclusión:

Como el p-valor es menor al nivel de significancia (0.0014 < 0.05), se rechaza la hipótesis nula, por ende, con un 95% de nivel de confianza podemos decir que la frecuencia de personas mayores de 80 años en la región metropolitana no es similar en las demás regiones.

Pregunta 2

La edad promedio de las personas mayores de 18 años ¿Es similar en los estados de soltero, casado y viudo?

  • Hipótesis nula (H₀): Las medias poblacionales de edad son iguales en los tres grupos de estado civil.

  • Hipótesis alternativa (Ha): Al menos una de las medias poblacionales de edad difiere entre los grupos.

Desarrollo de actividades para resolver la pregunta:

Se utilizará bootstrapping con remuestras en conjunto con ANOVA para contestar la pregunta planteada

1). Obtencion de datos e hipótesis a contrastar:

# Cargar sólo las variables necesarias
casen <- read_csv("EP08 Datos CASEN 2017.csv", col_types = cols()) |>
  select(id.vivienda, edad, ecivil) |>

niveles_ecivil <- c("Soltero(a)", "Casado(a)", "Viudo(a)")

casen <- casen |>

n_hogares <- 250

muestra2 <- casen |>

leveneTest(edad ~ ecivil, data = muestra2)

Como el p-valor del test de Levene es mayor al nivel de significancia (p>0.05), no se rechaza la hipótesis nula de homogeneidad de varianzas, por lo que se puede continuar con el análisis de ANOVA, con énfasis en el valor F observado.

# Estadístico observado
muestra2_anova <- muestra2 |>
  mutate(id = factor(1:n())) |>
  select(id, edad, ecivil)

resultado <- ezANOVA(
  data = muestra2_anova,
  dv = edad,
  wid = id,
  between = ecivil,
  type = 2
)

Con el fin de simular correctamente la hipótesis nula de igualdad de medias entre los grupos de estado civil, se realizo un reescalado de las edades dentro de cada grupo. Dicho procedimiento consistió en ajustar los valores de edad para que cada grupo tuviera la misma media y desviación estándar globales, manteniendo al mismo tiempo la estructura relativa interna de cada grupo. De esta forma, se genera un conjunto de datos artificial donde las diferencias entre grupos han sido eliminadas, permitiendo aplicar el método bootstrap de manera válida para contrastar la hipótesis nula.

grupos <- muestra2 |>
  group_by(ecivil) |>
  summarise(
    media = mean(edad),
    sd = sd(edad)
  ) |>
  as.data.frame()

grupos
##       ecivil    media       sd
## 1 Soltero(a) 43.27000 17.55733
## 2  Casado(a) 54.24648 15.51535
## 3   Viudo(a) 70.68889 13.61954
# Calcular media y desviación estándar global
media_gral <- mean(muestra2$edad)
sd_gral <- sd(muestra2$edad)


solteros <- muestra2 %>%
  filter(ecivil == "Soltero(a)") %>%
  mutate(edad = media_gral +
           (edad - mean(edad)) * (sd_gral / sd(edad)))

casados <- muestra2 %>%
  filter(ecivil == "Casado(a)") %>%
  mutate(edad = media_gral +
           (edad - mean(edad)) * (sd_gral / sd(edad)))

viudos <- muestra2 %>%
  filter(ecivil == "Viudo(a)") %>%
  mutate(edad = media_gral +
           (edad - mean(edad)) * (sd_gral / sd(edad)))

# Unir todos los grupos reescalados
muestra2_Z <- bind_rows(solteros, casados, viudos)

Ahora se generan las muestras bootstrap para cada grupo. Obteniendo valores de F para cada una de las muestras generadas, además se incluye un gráfico de la distribución de los valores de F obtenidos a partir de las muestras bootstrap, junto con el valor observado del estadístico F para obtener una comparación visual.

# Generacion de muestras bootstrap
set.seed(573)
B <- 2999

solteros <- muestra2_Z %>%
  filter(ecivil == "Soltero(a)") %>%
  select(edad, ecivil)

casados <- muestra2_Z %>%
  filter(ecivil == "Casado(a)") %>%
  select(edad, ecivil)

viudos <- muestra2_Z %>%
  filter(ecivil == "Viudo(a)") %>%
  select(edad, ecivil)


re_sol <- lapply(1:B, function(i) sample(nrow(solteros), replace = TRUE))
re_cas <- lapply(1:B, function(i) sample(nrow(casados),  replace = TRUE))
re_viu <- lapply(1:B, function(i) sample(nrow(viudos),  replace = TRUE))


# Función bootstrap para calcular el estadístico F
bootstrap_f <- function(b) {
  datos_b <- rbind(
    solteros[re_sol[[b]], ],
    casados[re_cas[[b]], ],
    viudos[re_viu[[b]], ]
  )
  
  modelo <- aov(edad ~ ecivil, data = datos_b)
  car::Anova(modelo, type = 2)$`F value`[1]
}



boot_F <- sapply(1:B, bootstrap_f)


p2 <- gghistogram(data.frame(boot_F), x = "boot_F",
                  title = "Distribución permutada",
                  xlab = "Estadístico F", ylab = "Frecuencia",
                  bins = 40, fill = "blue")
p2 <- p2 + geom_vline(xintercept = F_obs, colour="red")
print(p2)

En el gráfico,la línea roja vertical indica el valor de F observado en los datos reales. Dado que este valor se encuentra claramente fuera del rango esperado bajo la hipótesis nula, se estima que la diferencia observada entre los grupos es estadísticamente significativa.

Ahora se realizará el contraste de hipótesis, calculando el valor crítico y el p-valor empírico bootstrap. El valor crítico se obtiene a partir del percentil 95 de la distribución nula generada por el bootstrap, mientras que el p-valor empírico se calcula como la proporción de valores de F en la distribución nula que son mayores que el valor observado:

F_crit <- quantile(boot_F, 0.95)
num2 <- sum(boot_F > F_obs)
p_val2 <- (num2 + 1) / (B + 1)

print(F_crit)
##      95% 
## 3.121253
print(p_val2)
## [1] 0.0003333333

Como resultado, se obtuvo un valor crítico F igual a 3.12 (percentil 95% de la distribución nula), mientras que el valor del estadístico F observado fue superior a dicho umbral. Por otro lado el p-valor fue menor al nivel de significancia (p < 0.05), por lo tanto, se concluye con un 95% de confianza que existen diferencias significativas en la edad promedio entre al menos dos de los grupos de estado civil analizados (soltero, casado y viudo).

POST-HOC

En este caso, el resultado de la prueba fue estadísticamente significativo (p < 0.05), se realiza entonces el análisis post-hoc con fines académicos, para explorar posibles diferencias y ejercitar la metodología. Utilizando la diferencia de las medias como nuevo estadístico de interés.

Para cada par de grupos (soltero-casado, soltero-viudo, casado-viudo), se calculó la diferencia observada entre las medias de edad, y se generó una distribución nula de diferencias mediante 2999 remuestras bootstrap con reemplazo. A partir de estas distribuciones, se calcularon los valores-p empíricos bilaterales, los cuales darían un indicio para saber si las diferencias observadas son significativamente distintas de cero.

dif_obs_SC <- mean(solteros$edad) - mean(casados$edad)
dif_obs_SV <- mean(solteros$edad) - mean(viudos$edad)
dif_obs_CV <- mean(casados$edad)  - mean(viudos$edad)

get_dif_medias <- function(grupo1, grupo2, indices1, indices2) {
  media1 <- mean(grupo1[["edad"]][indices1])
  media2 <- mean(grupo2[["edad"]][indices2])
  media1 - media2
}

# Comparación Solteros - Casados
boot_dif_SC <- sapply(1:B, function(b) {
  get_dif_medias(solteros, casados, re_sol[[b]], re_cas[[b]])
})

# Comparación Solteros - Viudos
boot_dif_SV <- sapply(1:B, function(b) {
  get_dif_medias(solteros, viudos, re_sol[[b]], re_viu[[b]])
})

# Comparación Casados - Viudos
boot_dif_CV <- sapply(1:B, function(b) {
  get_dif_medias(casados, viudos, re_cas[[b]], re_viu[[b]])
})


# Diferencia observada y p-valor empírico bootstrap (dos colas)
dif_obs_SC <- mean(solteros$edad) - mean(casados$edad)
p_SC <- mean(abs(boot_dif_SC) >= abs(dif_obs_SC))
dif_obs_SV <- mean(solteros$edad) - mean(viudos$edad)
p_SV <- mean(abs(boot_dif_SV) >= abs(dif_obs_SV))
dif_obs_CV <- mean(casados$edad) - mean(viudos$edad)
p_CV <- mean(abs(boot_dif_CV) >= abs(dif_obs_CV))
