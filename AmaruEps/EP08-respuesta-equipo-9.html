Contexto

Como habamos visto a comienzos del semestre, la Encuesta de Caracterizacin Socioeconmica Nacional (Casen) es realizada por el Ministerio de Desarrollo Social de forma peridica para conocer la situacin de los hogares chilenos con relacin a aspectos demogrficos, de educacin, salud, vivienda, trabajo e ingresos. Es la principal fuente de informacin para estimar la magnitud de la pobreza y la distribucin del ingreso en el pas.

Se tiene a disposicin un archivo CSV con un subconjunto de los datos obtenidos en la Casen 2017. El equipo debe revisar las columnas disponibles en este archivo segn la descripcin en el libro de cdigos de la encuesta, que tambin queda disponible para este ejercicio. Es importante notar que en esta encuesta hay datos de carcter colectivo sobre el hogar del entrevistado, pero tambin hay datos de carcter individual, que se refieren al jefe o la jefa de hogar (no al entrevistado).

Hiptesis

  • \(H_0\): Las proporciones de personas con al menos educacin media completa son iguales entre la regin de Tarapac y la regin de Magallanes. Matemticamente: \(p_{tarapaca} - p_{magallanes} = 0\)

  • \(H_a\): Las proporciones de personas con al menos educacin media completa son distintas entre la regin de Tarapac y la regin de Magallanes. Matemticamente: \(p_{tarapaca} - p_{magallanes} \neq 0\)

Prueba de permutaciones Monte Carlo

n_perm <- 4500
alfa <- 0.05

dif_permutaciones <- replicate(n_perm, {
  # Mezclar los niveles educacionales
  datosFiltrados1$region_perm <- sample(datosFiltrados1$region)

  # Calcular la diferencia de proporciones
  prop_table_permutada <- datosFiltrados1 %>%
    group_by(region_perm) %>%
    summarise(
      prop_con_media = mean(nivel_educacional == "Con educacin media completa")
    )

  diff(prop_table_permutada$prop_con_media)
})

Se extrae el p-value y se grafica la distribucin nula de diferencias.

p_value <- mean(abs(dif_permutaciones) >= abs(dif_observada))
cat("p-value: ", round(p_value, 3))
## p-value:  0.494
hist(dif_permutaciones, breaks = 30, main = "Distribucin nula de diferencias", xlab = "Diferencia permutada")
abline(v = dif_observada, col = "red", lwd = 2)
abline(v = -abs(dif_observada), col = "blue", lty = 2)
abline(v = abs(dif_observada), col = "blue", lty = 2)

### Conclusin Como el p-value obtenido de 0.494 es mayor a 0.05, no se rechaza la hiptesis nula en favor de la alternativa. Por lo tanto, existe evidencia suficiente como para afirmar que con un 95% de confianza, las proporciones de personas con al menos educacin media completa son iguales entre la regin de Tarapac y la regin de Magallanes.

calcular_F_grupos <- function(grupo1, grupo2, grupo3) {
  datos_combinados <- bind_rows(grupo1, grupo2, grupo3)
  datos_combinados$estado_civil <- factor(datos_combinados$estado_civil, 
                                         levels = levels(datosPodados2$estado_civil))
  
  modelo <- aov(edad ~ estado_civil, data = datos_combinados)
  resumen <- summary(modelo)[[1]]
  return(resumen["estado_civil", "F value"])
}
F_verificacion <- calcular_F_grupos(datos_casado, datos_conviviente, datos_soltero)
