Contexto

Como habíamos visto a comienzos del semestre, la Encuesta de Caracterización Socioeconómica Nacional (Casen) es realizada por el Ministerio de Desarrollo Social de forma periódica para conocer la situación de los hogares chilenos con relación a aspectos demográficos, de educación, salud, vivienda, trabajo e ingresos. Es la principal fuente de información para estimar la magnitud de la pobreza y la distribución del ingreso en el país.

Se tiene a disposición un archivo CSV con un subconjunto de los datos obtenidos en la Casen 2017. El equipo debe revisar las columnas disponibles en este archivo según la descripción en el libro de códigos de la encuesta, que también queda disponible para este ejercicio. Es importante notar que en esta encuesta hay datos de carácter colectivo sobre “el hogar” del entrevistado, pero también hay datos de carácter individual, que se refieren “al jefe o la jefa de hogar” (no al entrevistado).

Hipótesis

  • \(H_0\): Las proporciones de personas con al menos educación media completa son iguales entre la región de Tarapacá y la región de Magallanes. Matemáticamente: \(p_{tarapaca} - p_{magallanes} = 0\)

  • \(H_a\): Las proporciones de personas con al menos educación media completa son distintas entre la región de Tarapacá y la región de Magallanes. Matemáticamente: \(p_{tarapaca} - p_{magallanes} \neq 0\)

Prueba de permutaciones Monte Carlo

n_perm <- 4500
alfa <- 0.05

dif_permutaciones <- replicate(n_perm, {
  # Mezclar los niveles educacionales
  datosFiltrados1$region_perm <- sample(datosFiltrados1$region)

  # Calcular la diferencia de proporciones
  prop_table_permutada <- datosFiltrados1 %>%
    group_by(region_perm) %>%
    summarise(
      prop_con_media = mean(nivel_educacional == "Con educación media completa")
    )

  diff(prop_table_permutada$prop_con_media)
})

Se extrae el p-value y se grafica la distribución nula de diferencias.

p_value <- mean(abs(dif_permutaciones) >= abs(dif_observada))
cat("p-value: ", round(p_value, 3))
## p-value:  0.494
hist(dif_permutaciones, breaks = 30, main = "Distribución nula de diferencias", xlab = "Diferencia permutada")
abline(v = dif_observada, col = "red", lwd = 2)
abline(v = -abs(dif_observada), col = "blue", lty = 2)
abline(v = abs(dif_observada), col = "blue", lty = 2)

### Conclusión Como el p-value obtenido de 0.494 es mayor a 0.05, no se rechaza la hipótesis nula en favor de la alternativa. Por lo tanto, existe evidencia suficiente como para afirmar que con un 95% de confianza, las proporciones de personas con al menos educación media completa son iguales entre la región de Tarapacá y la región de Magallanes.

Pregunta 2

Propongan una pregunta de investigación original, que involucre la comparación de las medias de más de dos grupos independientes (más abajo se dan unos ejemplos). Fijando una semilla distinta a la anterior, seleccionen una muestra aleatoria de hogares (200 < n < 300) y respondan la pregunta propuesta utilizando bootstrapping. Solo por ejercicio académico, aplique un análisis post-hoc con bootstrapping aunque este no fuera necesario. Indicación: consideren que el análisis post-hoc debe hacerse con las mismas remuestras usadas para la prueba ómnibus. Escrito matematicamente:

  • \(H_0: \mu_{casado} = \mu_{conviviente} = \mu_{soltero}\) (Las edades promedio son iguales entre los tres estados civiles)

  • \(H_a: \text{Al menos una } \mu_i \neq \mu_j \text{ para } i \neq j\) (Al menos un grupo de estado civil tiene una edad promedio distinta a los demás)

Respuesta

¿Son significativamente similares las edades de las personas según los estados civiles Casado(a), Conviviente o pareja sin acuerdo de unión civil y Soltero(a)?

calcular_F_grupos <- function(grupo1, grupo2, grupo3) {
  datos_combinados <- bind_rows(grupo1, grupo2, grupo3)
  datos_combinados$estado_civil <- factor(datos_combinados$estado_civil, 
                                         levels = levels(datosPodados2$estado_civil))
  
  modelo <- aov(edad ~ estado_civil, data = datos_combinados)
  resumen <- summary(modelo)[[1]]
  return(resumen["estado_civil", "F value"])
}
